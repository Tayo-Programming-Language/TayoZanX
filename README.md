# TayoZanX v1.0

![TayoZanX Logo](converted-image.png)

**TayoZanX** is an all-in-one beginner-friendly programming language designed for games, web, and apps. It combines scripting, markup, data, and UI routing in a simple syntax with an interpreter, compiler/VM, and DSL support.

---

## Features

- Beginner-friendly syntax
- Interpreter & Compiler/VM
- Game DSL (scene/entity/tick)
- UI / Web DSL with routing
- Online playground generator
- VS Code extension support
- Spec book (`TayoZanX_Spec.txt`)
- Safe mode for shell commands
